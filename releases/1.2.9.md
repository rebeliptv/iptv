## What's New

- **FIFA World Cup & international soccer** now appear alongside your other live sports, with team crests and colors.
- Live channels keep playing seamlessly when their stream source switches or dies behind the scenes — no more error-and-restart or visible rebuffer.
- Channels only show as available when the feed actually has something to play (not just a reachable address), so dead channels stop cluttering your lineup.
- Your custom M3U sources are always validated and kept in your playlist and guide, with plain-language hints when one can't load.
- New first-run setup wizard walks you through picking a source, adding an optional API key, connecting Jellyfin or Plex, and optionally setting a dashboard login.
- Optionally **password-protect the dashboard** — set an admin username/password in setup (or skip to leave it open on your network); sign out from the top bar. Manage multiple accounts under Settings → Dashboard Login (add users, reset passwords, remove accounts). Removing the last account turns login back off and reopens the dashboard on your network.
- Refresh now shows a labelled progress bar instead of an endless spinner, plus an added channel source for broader coverage.
- The dashboard now reacts to a refresh instantly — the Status card reads "Refreshing" with live progress and the counts climb as it runs, instead of looking idle until you reload the page.

## Docker

```bash
docker pull rebeliptv/iptv:1.2.9
```

### Docker Compose

```yaml
services:
  iptv:
    image: rebeliptv/iptv:1.2.9
    hostname: iptv
    container_name: iptv
    ports:
      - "8080:8080"
    # Environment variables are OPTIONAL — defaults work out of the box.
    # Uncomment any you need:
    # environment:
    #   TZ: "America/New_York"   # set to YOUR timezone, e.g. Europe/London, Etc/UTC
    #   USE_PROXY: "true"        # only if providers are blocked in your region
    volumes:
      - iptv-data:/app/data
    dns:
      - 8.8.8.8
      - 4.2.2.2
      - 1.1.1.1
    stop_grace_period: 30s
    restart: unless-stopped

volumes:
  iptv-data:
```

### Docker Run

```bash
docker run -d \
  --name iptv \
  -p 8080:8080 \
  -v iptv-data:/app/data \
  --dns 8.8.8.8 \
  --dns 4.2.2.2 \
  --dns 1.1.1.1 \
  --restart unless-stopped \
  rebeliptv/iptv:1.2.9

# Environment variables are optional (defaults work). Add -e flags as needed, e.g.:
#   -e TZ=America/New_York   (set to YOUR timezone — e.g. Europe/London, Etc/UTC)
#   -e USE_PROXY=true        (only if providers are blocked in your region)
```

### Updating

```bash
docker pull rebeliptv/iptv:1.2.9
docker compose up -d
```

## Endpoints

| URL | Description |
|-----|-------------|
| `http://<server>:8080/playlist?key=YOUR_KEY` | M3U playlist |
| `http://<server>:8080/epg?key=YOUR_KEY` | EPG guide (XML) |
| `http://<server>:8080/epg.gz?key=YOUR_KEY` | EPG guide (gzip) |
| `http://<server>:8080` | Web dashboard |

> **Note:** The `?key=` parameter is only required if an API key has been generated in the web dashboard under **Settings**.
