## What's New

- Local broadcast stations now show their city, state, and network in the channel name — e.g. **WABC New York NY (ABC)** instead of just **WABC** — so you can find a local channel by city or network in your player's search instead of having to know its call sign.
- You can now filter the channel list by country (US, Canada, UK) in the dashboard.
- Channel search now matches the network and city too, not just the name — search **FOX**, **NBC**, **CBS**, or **ABC** to get every local affiliate of that network, or a city like **New York** to find the locals there.
- You can now enable/disable channels and **drag to reorder** them from the dashboard. Open **Settings → Channel Lineup** to drop channels you don't watch from your Jellyfin/Plex playlist and guide, or drag rows to set the order — e.g. pull your local stations to the top. Disabled channels disappear from the channels list, guide, and counts (and stop being scraped); manage them on the lineup page, where **Show disabled** reveals them at the bottom. The lineup lists every supported channel (so you can manage ones that are currently offline too), and an **Actions** menu lets you reset the order to default (keeping your on/off choices) or enable/disable channels in bulk — filter to a category first to flip just that group.
- The **Settings** page is now organized into sections — **Server**, **Channels**, **Access**, and **Player** — with a sidebar, instead of one long scroll.

## Docker

```bash
docker pull rebeliptv/iptv:1.2.13
```

### Docker Compose

```yaml
services:
  iptv:
    image: rebeliptv/iptv:1.2.13
    hostname: iptv
    container_name: iptv
    ports:
      - "8080:8080"
    # Environment variables are OPTIONAL — defaults work out of the box.
    # Uncomment any you need:
    # environment:
    #   TZ: "America/New_York"   # set to YOUR timezone, e.g. Europe/London, Etc/UTC
    volumes:
      - iptv-data:/app/data
    dns:
      - 8.8.8.8
      - 4.2.2.2
      - 1.1.1.1
    stop_grace_period: 30s
    restart: unless-stopped

volumes:
  iptv-data:
```

### Docker Run

```bash
docker run -d \
  --name iptv \
  -p 8080:8080 \
  -v iptv-data:/app/data \
  --dns 8.8.8.8 \
  --dns 4.2.2.2 \
  --dns 1.1.1.1 \
  --restart unless-stopped \
  rebeliptv/iptv:1.2.13

# Environment variables are optional (defaults work). Add -e flags as needed, e.g.:
#   -e TZ=America/New_York   (set to YOUR timezone — e.g. Europe/London, Etc/UTC)
```

### Updating

```bash
docker pull rebeliptv/iptv:1.2.13
docker compose up -d
```

## Endpoints

| URL | Description |
|-----|-------------|
| `http://<server>:8080/playlist?key=YOUR_KEY` | M3U playlist |
| `http://<server>:8080/epg?key=YOUR_KEY` | EPG guide (XML) |
| `http://<server>:8080/epg.gz?key=YOUR_KEY` | EPG guide (gzip) |
| `http://<server>:8080` | Web dashboard |

> **Note:** The `?key=` parameter is only required if an API key has been generated in the web dashboard under **Settings**.
