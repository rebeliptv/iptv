## What's New

- Added the **UK Sky Sports** channels — Main Event, Premier League, Football, Cricket, F1, Golf, News, Action, Arena, Mix, Racing, Tennis, NFL and Darts — with programme guide data (all except Arena, which streams without a guide).
- Added extra backup sources for popular US channels (ABC, CBS, CNN, ESPN, FOX, HBO and more) and for live sports, so they keep playing when one provider has trouble.
- Fixed certain channels not playing in the dashboard's in-browser preview — they now play there just like they already did in VLC / Plex / Jellyfin.
- Smoother, more reliable playback — less buffering and faster starts, quicker switches to a backup source when one has trouble, and audio no longer cuts out after buffering or a source change.
- Fixed the dashboard occasionally failing to load — most often the live Events view — when the app was busy.
- The Channels page now loads in pages (28 at a time, with larger options) so big channel lists stay fast.
- Clearer step-by-step progress while a refresh is running.

## Docker

```bash
docker pull rebeliptv/iptv:1.2.10
```

### Docker Compose

```yaml
services:
  iptv:
    image: rebeliptv/iptv:1.2.10
    hostname: iptv
    container_name: iptv
    ports:
      - "8080:8080"
    # Environment variables are OPTIONAL — defaults work out of the box.
    # Uncomment any you need:
    # environment:
    #   TZ: "America/New_York"   # set to YOUR timezone, e.g. Europe/London, Etc/UTC
    #   USE_PROXY: "true"        # only if providers are blocked in your region
    volumes:
      - iptv-data:/app/data
    dns:
      - 8.8.8.8
      - 4.2.2.2
      - 1.1.1.1
    stop_grace_period: 30s
    restart: unless-stopped

volumes:
  iptv-data:
```

### Docker Run

```bash
docker run -d \
  --name iptv \
  -p 8080:8080 \
  -v iptv-data:/app/data \
  --dns 8.8.8.8 \
  --dns 4.2.2.2 \
  --dns 1.1.1.1 \
  --restart unless-stopped \
  rebeliptv/iptv:1.2.10

# Environment variables are optional (defaults work). Add -e flags as needed, e.g.:
#   -e TZ=America/New_York   (set to YOUR timezone — e.g. Europe/London, Etc/UTC)
#   -e USE_PROXY=true        (only if providers are blocked in your region)
```

### Updating

```bash
docker pull rebeliptv/iptv:1.2.10
docker compose up -d
```

## Endpoints

| URL | Description |
|-----|-------------|
| `http://<server>:8080/playlist?key=YOUR_KEY` | M3U playlist |
| `http://<server>:8080/epg?key=YOUR_KEY` | EPG guide (XML) |
| `http://<server>:8080/epg.gz?key=YOUR_KEY` | EPG guide (gzip) |
| `http://<server>:8080` | Web dashboard |

> **Note:** The `?key=` parameter is only required if an API key has been generated in the web dashboard under **Settings**.
